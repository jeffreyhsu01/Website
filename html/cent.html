Every Cent Counts

An application packaged with Docker that rounds every purchase up to the nearest dollar and donates the difference to charity!

I'm developing this project using the PERN Stack with PostgreSQL in conjecture with the aid of Plaid API to simulate banking and transactional information.

Still a work in progress so check back shortly! :)
