Beat the Bookie...Hypothetically

I’ve always believed that analytics is crap.

- Charles Barkley

A passion project combining basketball with machine learning! I’m gonna be honest this project will take a while so check back shortly!

Currently using a custom algorithm factoring player chemistry to help account for what analytics cannot show. The algorithm is housed through Docker and published to Amazon ECR.

The data is gathered using Beautiful Soup and the model is trained and deployed using PyTorch, Amazon Sagemaker, and Amazon S3.
